# 网店发票填写器协作约定

## 项目边界

- 本项目是 Chrome 扩展，用于把网店账单截图经 OCR 识别后，辅助填入厦门电子税务局蓝字发票开具页面。
- 插件只做识别和填表辅助，不自动点击开票、提交、上传发票等最终确认按钮。
- MiniMax API Key 等敏感信息只允许放在本地 `.env`，不能写入前端代码、README、开发日志或 Git 提交。

## 开发日志

- 每次完成本项目的功能开发、Bug 修复、打包验证、发布推送或重要调试后，都要更新项目开发日志。
- 开发日志目录：`/Users/gd/Library/Mobile Documents/iCloud~md~obsidian/Documents/项目管理/小项目/网店发票填写器/开发/`
- 开发日志索引：`/Users/gd/Library/Mobile Documents/iCloud~md~obsidian/Documents/项目管理/小项目/网店发票填写器/开发/开发日志索引.md`
- 记录方式：按天一个文件，文件名固定为 `YYYY-MM-DD.md`。
- 如果当天已有日期文件，追加或合并到同一文件；不要重复创建同一天日志。
- 日期文件使用这些章节：`今日概览`、`关键记录`、`测试与验证`、`文档与资源`、`后续处理`、`本次采集情况`。
